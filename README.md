\## Description

This is an interactive \*\*Tic-Tac-Toe game\*\* built for two players.  

Players take turns to place \*\*X\*\* or \*\*O\*\* on the grid. The game tracks moves and automatically detects the winner or a draw.  



The application is designed with a \*\*dark tech theme\*\* to give a modern and professional look.



\## Features

\- 3x3 interactive \*\*Tic-Tac-Toe grid\*\*  

\- \*\*Player turn tracking\*\* (X and O)  

\- \*\*Winner detection\*\* and draw handling  

\- \*\*Restart button\*\* to reset the game  

\- \*\*Dark tech design\*\* with hover effects and glowing text  



\## Technologies Used

\- HTML5  

\- CSS3 (with animations and dark theme)  

\- JavaScript (DOM manipulation, event handling, game logic)  



\## How to Run

1\. Clone or download the repository  

2\. Open `index.html` in a web browser  

3\. Click on a cell to make your move  

4\. Use the \*\*Restart Game\*\* button to start over
